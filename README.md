## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Mindfull is an app that helps users reduce procrastination and executive dysfunction. Users set tasks for the day, and after reaching app time limits, AI breaks these tasks into small, actionable steps. Completing these steps allows users to “earn back” increments of screen time, combining productivity with digital wellbeing.

The app is designed with accessibility and ADHD-friendly task management in mind.
